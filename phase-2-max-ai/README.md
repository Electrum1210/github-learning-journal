## Overview

Learn to work with Max (AI operational co-founder) to amplify your effectiveness as COO. This training focuses on operational leverage through AI, not technical depth.

**What you'll learn:**
- When and how to use Max for operational tasks
- Effective prompting for GitHub-Notion workflows
- Technical decision support without coding
- Documentation automation
- Building custom operational workflows

---

## Training Objectives

### Primary Goal
Leverage Max for operational tasks, GitHub-Notion workflows, and technical decision support without requiring deep coding skills.

### Key Skills
1. **Understanding Max's capabilities** - What Max can/can't do
2. **Effective prompting** - Structure requests for optimal responses
3. **GitHub-Notion sync** - Automate project tracking and updates
4. **Technical support** - Code review assistance and risk assessment
5. **Documentation** - Auto-generate and maintain project docs
6. **Task delegation** - When to use Max vs escalate to humans

---

## Learning Approach

**Manager-first perspective:**
- Not learning to code
- Learning to leverage AI as an operational tool
- Focus on practical workflows, not AI theory
- ADHD-optimized: clear, actionable, immediate feedback

**Practice-based:**
- Real DeDynamics scenarios
- Working with actual repos and Notion pages
- Iterative refinement of prompts
- Building custom workflows
